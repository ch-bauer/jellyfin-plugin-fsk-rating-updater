FSK Rating Updater

Free key from themoviedb.org (Settings → API). Leave empty to only normalize existing ratings without any online lookups.
Only log the proposed changes (see Dashboard → Logs) without writing anything.
Seasons and episodes use the series certification from TMDb and otherwise inherit the rating from their parent. Requires "Update TV series".
When off, an existing FSK rating is only reformatted (e.g. "FSK 12" → "FSK-12"). When on, TMDb is asked again and may overwrite it.
